# VaxLog

A simple way to keep track of all of your pets vaccines.

### Installing
To use VaxLog:

1. Clone https://github.com/PBillingsby/sinatra-project-flatiron.git and change into directory.
2. Run ```bundle install``` in terminal.
3. Run ```rake db:migrate```
4. Run ```shotgun```
